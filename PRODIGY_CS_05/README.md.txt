### Description
Captures and analyzes network packets, displaying key details such as source/destination IPs, protocols, and payload data.

### Features
- **Packet Capture**: Uses the `scapy` library for packet sniffing.
- **Detailed Analysis**: Displays protocol and IP information.
- **Error Handling**: Handles permissions and interruptions smoothly.

### Usage
```bash
# Install npcap (required)
# Run the script
python packet_analyzer.py
```

### Requirements
- `scapy`

### Additional Information
- Download and install `npcap` from [npcap.org](https://npcap.org) with pcap mode enabled.
